# Personvernerklæring for Svario

Status: arbeidsutkast. Må juridisk gjennomgås før publisering.

Sist oppdatert: 21. juni 2026

## 1. Hvem er ansvarlig?

Svario leveres av Kapsdevelopment AS, org.nr. 937 284 624.

Nettside: https://kapsdevelopment.com/

Kontaktpunkt: kontaktskjemaet på https://kapsdevelopment.com/. Skjemaet skal kunne brukes til spørsmål om personvern, sikkerhet, innsyn, retting, sletting og andre rettigheter. Henvendelser rutes internt til riktig mottaker.

For spørreskjemaer er det vanligvis skjemaeieren som er behandlingsansvarlig for formål, innhold, rettslig grunnlag og lagringstid. Svario er databehandler for besvarelsene.

For Svario-kontoer, drift, sikkerhet, avtaleoppfølging og administrasjon er Kapsdevelopment AS behandlingsansvarlig.

## 2. Hva samler Svario inn?

Svario kan behandle:

- kontoopplysninger for administratorer,
- profil- og arbeidsflateopplysninger,
- spørreskjemaer, spørsmål og svar,
- respondentnavn, e-postadresse eller lignende dersom skjemaeier velger identifisert skjema,
- fritekst som respondenter skriver inn,
- samtykke- og personvernvalg der skjemaet krever det,
- tekniske opplysninger som er nødvendige for drift, sikkerhet og feilretting,
- henvendelser sendt via kontaktskjema.

Svario lagrer ikke IP-adresse på spørreskjemabesvarelser i appens besvarelsestabeller.

Tekniske logger hos hosting-, database- eller sikkerhetsleverandører kan likevel forekomme for drift, sikkerhet og feilretting. Slike logger skal ikke brukes til analyse av respondenter, markedsføring eller salg av data.

## 3. Hva bruker Svario opplysningene til?

Svario bruker opplysninger for å:

- levere spørreskjematjenesten,
- autentisere og administrere brukere,
- lagre og vise resultater til autoriserte brukere,
- eksportere rapporter etter brukerens valg,
- gjennomføre automatisk sletting etter valgt lagringstid,
- dokumentere relevante personvernhendelser,
- svare på henvendelser,
- sikre, drifte og feilrette tjenesten.

Svario bruker ikke spørreskjemabesvarelser til annonsering, profilering, salg av data eller trening av modeller.

## 4. Rettslig grunnlag

For kundens spørreskjemaer velger og dokumenterer kunden rettslig grunnlag. Svario legger til rette for at kunden kan beskrive formål, rettslig grunnlag, kontaktpunkt og lagringstid for hvert skjema.

For Svario sine egne behandlingsformål kan rettslig grunnlag blant annet være:

- avtale, når behandlingen er nødvendig for å levere tjenesten,
- berettiget interesse, for eksempel drift, sikkerhet, feilretting og misbruksforebygging,
- rettslig plikt, for eksempel regnskaps- eller dokumentasjonskrav,
- samtykke, der dette er relevant og frivillig.

## 5. Lagringstid

Besvarelser lagres etter lagringstiden skjemaeier velger for skjemaet. Når lagringstiden utløper, slettes svar automatisk når retention-jobben kjører.

Dersom skjemaeier forlenger lagringstiden for et skjema med eksisterende svar, må skjemaeier oppgi en begrunnelse. Forlengelsen logges.

Konto-, kontakt- og driftsopplysninger lagres så lenge det er nødvendig for å levere tjenesten, følge opp henvendelser, oppfylle rettslige plikter, håndtere sikkerhet eller dokumentere avtaleforhold.

## 6. Deling og underdatabehandlere

Svario bruker underdatabehandlere for drift, database, autentisering, hosting og deploy. Oversikt finnes i `underdatabehandlere.md`.

Svario selger ikke personopplysninger og deler ikke besvarelser med tredjeparter for kommersielle formål.

## 7. Dine rettigheter

Registrerte kan ha rett til innsyn, retting, sletting, begrensning, dataportabilitet og protest etter GDPR.

For svar i et kundestyrt skjema bør henvendelser normalt rettes til skjemaeieren, fordi skjemaeieren bestemmer formål og lagringstid. Svario kan bistå skjemaeieren ved behov.

Hvis henvendelsen gjelder Svario-konto, drift, sikkerhet eller Kapsdevelopment AS sin egen behandling, kan kontaktskjemaet på https://kapsdevelopment.com/ brukes.

## 8. Sikkerhet

Svario bruker tilgangsstyring, Row Level Security/eierkontroller, kryptert transport, leverandørstyrt kryptering ved lagring, dataminimering, logging av relevante personvernhendelser og automatisk sletting for å beskytte data.

Svario er ikke ISO 27001-sertifisert. Tekniske og organisatoriske tiltak beskrives i `tekniske-og-organisatoriske-tiltak.md`.
