# Personvernerklæring for Svario

Status: arbeidsutkast. Må juridisk gjennomgås for publisering.

Sist oppdatert: 20. juni 2026

## 1. Hvem er ansvarlig?

Svario leveres av `[JURIDISK NAVN]`, org.nr. `[ORG.NR]`.

Kontakt: `[E-POST]`.

For spørreskjemaer er det vanligvis skjemaeieren som er behandlingsansvarlig for formål, innhold og lagringstid. Svario er databehandler for besvarelsene.

For Svario-kontoer, drift, sikkerhet og administrasjon er Svario behandlingsansvarlig for egne behandlingsformål.

## 2. Hva samler Svario inn?

Svario kan behandle:

- kontoopplysninger for administratorer,
- profil- og arbeidsflateopplysninger,
- spørreskjemaer, spørsmål og svar,
- respondentnavn eller e-post dersom skjemaeier velger identifisert skjema,
- fritekst som respondenter skriver inn,
- tekniske opplysninger som er nødvendige for drift, sikkerhet og feilretting.

Svario lagrer ikke IP-adresse på spørreskjemabesvarelser i appens besvarelsestabeller.

## 3. Hva bruker Svario opplysningene til?

Svario bruker opplysninger for å:

- levere spørreskjematjenesten,
- autentisere og administrere brukere,
- lagre og vise resultater,
- eksportere rapporter etter brukerens valg,
- gjennomføre automatisk sletting etter valgt lagringstid,
- sikre, drifte og forbedre tjenesten.

Svario bruker ikke spørreskjemabesvarelser til annonsering, profilering, salg av data eller trening av modeller.

## 4. Rettslig grunnlag

For kundens spørreskjemaer velger og dokumenterer kunden rettslig grunnlag.

For Svario sine egne behandlingsformål kan rettslig grunnlag blant annet være avtale, berettiget interesse, rettslig plikt eller samtykke der det er relevant.

## 5. Lagringstid

Besvarelser lagres etter lagringstiden skjemaeier velger for skjemaet og slettes automatisk når retention-jobben kjører.

Konto- og driftsopplysninger lagres så lenge det er nødvendig for å levere tjenesten, oppfylle rettslige plikter, håndtere sikkerhet eller dokumentere avtaleforhold.

## 6. Deling og underdatabehandlere

Svario bruker underdatabehandlere for drift, database, autentisering og hosting. Oversikt finnes i `underdatabehandlere.md`.

Svario selger ikke personopplysninger og deler ikke besvarelser med tredjeparter for kommersielle formål.

## 7. Dine rettigheter

Registrerte kan ha rett til innsyn, retting, sletting, begrensning, dataportabilitet og protest etter GDPR.

For svar i et kundestyrt skjema bør henvendelser normalt rettes til skjemaeieren, fordi skjemaeieren bestemmer formål og lagringstid. Svario kan bistå skjemaeieren ved behov.

## 8. Sikkerhet

Svario bruker tilgangsstyring, RLS/eierkontroller, kryptert transport, plattformkryptering ved lagring, dataminimering og automatisk sletting for å beskytte data.

## 9. Kontakt og klage

Kontakt Svario på `[E-POST]`.

Du kan også klage til Datatilsynet dersom du mener personopplysninger behandles i strid med regelverket.
