# Juridiske arbeidsutkast

Dette området samler arbeidsutkast for Svario sine personvern- og databehandlerdokumenter. Tekstene er laget for produkt- og implementasjonsarbeid, og må gjennomgås av juridisk kompetanse før de publiseres eller brukes som bindende avtale.

## Dokumenter

- `databehandleravtale.md`: utkast til databehandleravtale mellom skjemaeier og Svario.
- `personvernerklæring.md`: utkast til personvernerklæring for Svario som tjeneste.
- `underdatabehandlere.md`: oversikt over underdatabehandlere og formål.

## Må fylles ut for publisering

- Juridisk navn på Svario-leverandøren.
- Organisasjonsnummer.
- Kontaktadresse og personvernkontakt.
- Endelig liste over underdatabehandlere, regioner og lenker til deres databehandleravtaler.
- Rutine for håndtering av innsyn, sletting, eksport, avvik og kundens instruksjoner.
- Versjonsdato og godkjenningsansvarlig.

## Kilder brukt som struktur

- GDPR artikkel 28 om databehandleravtaler.
- GDPR artikkel 13 og 14 om informasjon til registrerte.
- Datatilsynet sine veiledningssider om behandlingsgrunnlag og virksomheters plikter.
